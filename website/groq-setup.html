Setup Guide

Set up Polishly with
a free Groq key.

Unlock lightning-fast rewrites across your Mac by pairing Polishly with Groq's generous free tier.

Step-by-step instructions

1. Get your free Groq API key

Groq offers a generous free tier for developers with high rate limits, perfect for quick rewrites.

  • Go to console.groq.com/keys.
  • Sign in or create a free account (no credit card required).
  • Click Create API Key.
  • Give it a name like "Polishly Mac App".
  • Copy the generated key (it starts with gsk_). You won't be able to see it again.

2. Configure Polishly

Now, tell Polishly to use your new key.

  • Click the Polishly icon in your Mac's menu bar.
  • Select Settings....
  • Under AI Provider, choose Groq.
  • In the API Key field, paste the gsk_... key you just copied.
  • By default, Polishly uses llama3-70b-8192. You can change this to llama3-8b-8192 if you want even faster results.
  • Click Save.

3. Try it out

You're ready to go!

  • Select some text in any app (like Notes or Mail).
  • Press the hotkey: ⌃⌥Space.
  • Watch the rewrite stream in instantly.
